# note

> A Vue.js project

## 技术栈

> vue2 + vuex + vue-router + webpack + ES6 sass + element-ui


## 项目运行

``` bash
cd note

npm install  //依赖安装

npm run dev  //运行项目

npm run build  //压缩项目（可不执行）
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
